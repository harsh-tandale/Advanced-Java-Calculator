# Advanced Calculator in Java

## Description
This is a beginner-friendly advanced calculator built in Java.
It supports basic arithmetic operations using a menu-driven approach.

## Features
- Addition, Subtraction, Multiplication, Division
- Uses methods and switch-case
- Loop-based execution
- Handles division by zero

## Technologies Used
- Java
- Scanner class

## How to Run
1. Compile: javac AdvancedCalculator.java
2. Run: java AdvancedCalculator
